# How to Type Katakana on Different Devices

Katakana is the go-to script for foreign loanwords, technical terms, onomatopoeia, and emphasis in Japanese. Whether you want to type コーヒー (koohii, coffee), スマートフォン (sumaatofon, smartphone), or ティーム (timu, team), knowing how to input katakana quickly makes a real difference. This guide focuses on katakana-specific input techniques — including the conversion shortcuts most learners never discover.

## Setting Up Japanese Input

Before you can type any katakana, you need a Japanese IME (Input Method Editor) installed on your device. An IME lets you type in romaji (Roman letters) and converts them to Japanese characters.

### Quick Setup Summary

| Platform | IME | Install Method |
|----------|-----|----------------|
| Windows | Microsoft IME | Settings → Time & Language → Language & Region → Add Japanese |
| macOS | Apple Japanese Input | System Settings → Keyboard → Input Sources → Add Japanese |
| iPhone/iPad | Built-in | Settings → General → Keyboard → Keyboards → Add Japanese |
| Android | Gboard | Play Store → search "Gboard" → add Japanese in app settings |

Once Japanese input is installed, you type romaji and the IME produces hiragana by default. The key skill for katakana is knowing how to convert that hiragana into katakana — either before you commit the text or by switching to a direct katakana mode.

To understand which characters you are working with, the [Hiragana-Katakana Chart](/hiragana-katakana-chart) is a useful reference for the full katakana syllabary alongside its hiragana counterparts.

## Typing Katakana on Windows (Microsoft IME)

### The F7 Shortcut — The Fastest Route to Katakana

The single most important key for katakana on Windows is **F7**. After typing a word in romaji (while the text is still underlined and unconverted), pressing F7 converts it directly to full-width katakana.

**Step-by-step:**
1. Switch the IME to Japanese mode (click the taskbar icon showing `A` and switch to `あ`, or press `Alt + ~`)
2. Type romaji — for example: `koohii`
3. The IME shows underlined hiragana: こうひい (before you press Space)
4. Press **F7** instead of Space
5. The text converts to katakana: コーヒー

No manual Space bar cycling through conversion candidates required. F7 jumps straight to katakana.

### Windows Conversion Shortcut Keys

| Key | Result | Example Input → Output |
|-----|--------|------------------------|
| F6 | Full-width hiragana | `ko` → こ |
| **F7** | **Full-width katakana** | `ko` → **コ** |
| F8 | Half-width katakana | `ko` → ｺ |
| F9 | Full-width romaji | `ko` → ｋｏ |
| F10 | Half-width romaji | `ko` → ko |

You can press F7 multiple times while text is selected to cycle through katakana sub-variants (full-width → half-width → romaji).

### Direct Katakana Input Mode

If you are typing a passage that is entirely in katakana — say, a list of product names or foreign vocabulary — switching to direct katakana mode is more efficient than pressing F7 after every word.

**Activate direct katakana mode:**
1. Right-click the IME icon in the taskbar
2. Select **Katakana** (カタカナ) from the context menu
3. Everything you type in romaji now produces katakana directly

To return to hiragana mode, right-click the IME icon again and select **Hiragana** (ひらがな), or press `Alt + ~` to toggle.

**Keyboard shortcut for mode switching (Microsoft IME):**
- `Ctrl + Caps Lock` — switch to hiragana input
- `Alt + Caps Lock` — switch to katakana input

### Typing After F7 — Confirming the Conversion

After pressing F7, the katakana text is still in a pending state (underlined). Press **Enter** to commit it to the document. If you want to adjust the selection boundary — for example, to split a compound word — use the left/right arrow keys while the text is underlined.

## Typing Katakana on Mac

### F7 and Control+K — Two Shortcuts, One Goal

macOS supports the same F7 function key shortcut as Windows, but many Mac keyboards require pressing **Fn + F7** because the top row defaults to media controls. Check your keyboard settings: if "Use F1, F2, etc. keys as standard function keys" is enabled in System Settings → Keyboard, you can press F7 directly.

The second shortcut is **Control + K**, which is unique to macOS's Japanese input and is often more convenient because it does not require the Fn key.

**Both shortcuts do the same thing:** convert the current pending hiragana to katakana.

### macOS Conversion Shortcut Keys

| Shortcut | Result |
|----------|--------|
| **F7** (or Fn+F7) | Full-width katakana |
| **Control + K** | Full-width katakana |
| Control + J | Hiragana |
| Control + L | Full-width romaji |
| Control + ; | Half-width romaji |

**Example using Control+K:**
1. Switch to Japanese input (click the menu bar icon → Hiragana, or use your configured shortcut)
2. Type: `sumaatofon`
3. IME shows pending hiragana: スマートフォン ... actually the IME shows it in hiragana first
4. Press **Control + K** before pressing Space or Return
5. Output: スマートフォン

### Direct Katakana Mode on Mac

macOS does not expose a simple right-click menu to switch to katakana mode the same way Windows does, but you can:

1. Click the input menu in the menu bar (the `A` or `あ` icon)
2. Select **Katakana** if it appears in the menu

If the katakana option is not visible, go to **System Settings → Keyboard → Input Sources**, select the Japanese input source, and enable the katakana input option.

Alternatively, many Mac users simply rely on Control+K after each word rather than switching modes.

## Typing Katakana on iPhone and Android

### iPhone and iPad

iOS does not have function keys, so katakana conversion works differently. When you type in the Japanese Romaji keyboard and text is pending in the suggestion bar, you can:

1. Type the word — for example: `terebi`
2. Look at the suggestion bar above the keyboard
3. Slide the suggestion bar left to find katakana candidates: テレビ
4. Tap the katakana version

For words that are inherently foreign loanwords (and are therefore almost always written in katakana), the IME usually surfaces the katakana version near the top of suggestions.

**Japanese Kana keyboard (flick input):**
- If you use the flick keyboard instead of romaji, there is a `カナ` key that switches the keyboard to katakana input mode directly
- Tap `カナ` to enter katakana mode, type, then tap `かな` to return to hiragana

### Android (Gboard)

Gboard's Japanese mode works similarly. After typing romaji:

1. Type: `anime`
2. The suggestion bar shows candidates including アニメ (katakana) alongside 亜仁目 and others
3. Tap アニメ to insert it

For repeated katakana input in Gboard, long-press the suggestion bar entry or look for the katakana toggle in the keyboard's language settings panel.

## Typing Special Katakana Characters

Katakana has several characters that require specific romaji combinations or dedicated key sequences. These come up constantly when typing foreign loanwords.

### The Long Vowel Mark ー (Choonpu)

The long vowel mark ー extends a vowel sound and appears in almost every loanword. For example: コーヒー (koohii, coffee), ケーキ (keeki, cake), スーパー (suupaa, supermarket).

| Input method | What to type |
|--------------|--------------|
| Windows / macOS | `-` (hyphen key) while in Japanese mode |
| iOS romaji keyboard | `-` (hyphen) |
| Android Gboard | `-` (hyphen) |

The hyphen key is mapped to ー in all major IMEs. You do not type a doubled vowel to get ー in katakana — you type a hyphen.

**Compare hiragana vs katakana long vowels:**
- Hiragana: おかあさん (okaasan) — the long vowel is written as an extra vowel character
- Katakana: コーヒー (koohii) — the long vowel is always written as ー

### Small ッ (Small Tsu — Double Consonant)

Small ッ represents a brief pause or doubled consonant, the same way small っ does in hiragana. Type the following consonant twice.

| Word | Romaji to type | Result |
|------|---------------|--------|
| ベッド (beddo, bed) | `beddo` | ベッド |
| カップ (kappu, cup) | `kappu` | カップ |
| ネット (netto, internet/net) | `netto` | ネット |
| チケット (chiketto, ticket) | `chiketto` | チケット |

To type a standalone small ッ, type `xtsu` or `ltsu` (the `x` or `l` prefix forces a small character in most IMEs).

### Foreign Sound Combinations

Katakana has extended combinations to represent sounds that do not exist in native Japanese phonology. These are used for foreign loanwords and require specific romaji input.

| Katakana | Romaji | Example word |
|----------|--------|--------------|
| ティ | `ti` or `thi` | ティーム (timu, team) |
| ディ | `di` or `dhi` | ディスコ (disuko, disco) |
| ファ | `fa` | ファイル (fairu, file) |
| フィ | `fi` | フィルム (firumu, film) |
| フェ | `fe` | フェア (fea, fair) |
| フォ | `fo` | フォーム (foumu, form) |
| ウィ | `wi` | ウィルス (wirusu, virus) |
| ウェ | `we` | ウェブ (webu, web) |
| ヴァ | `va` | ヴァイオリン (vaiorin, violin) |
| チェ | `che` | チェック (chekku, check) |
| ジェ | `je` | ジェット (jetto, jet) |
| トゥ | `twu` or `tu` | トゥルー (turuu, true) |
| ドゥ | `dwu` or `du` | ドゥーム (duumu, doom) |

**Note on ティ vs チ:** These are different sounds. チ (chi) is the traditional Japanese sound. ティ (ti) represents the English "ti" sound as in "team" or "tip," and is used in modern loanwords to preserve the original pronunciation.

### Small Katakana Vowels (for Extended Sounds)

Some loanwords use standalone small katakana vowels for unusual combinations:

| Katakana | Romaji | Usage |
|----------|--------|-------|
| ァ (small a) | `xa` or `la` | Rare, used in some extended combinations |
| ィ (small i) | `xi` or `li` | As in ティ, ディ combinations |
| ゥ (small u) | `xu` or `lu` | Rare combinations |
| ェ (small e) | `xe` or `le` | As in チェ, ジェ combinations |
| ォ (small o) | `xo` or `lo` | As in フォ combinations |

Most of the time you will not need to type these individually — the full combination romaji (like `fa` for ファ) produces the correct output automatically.

## Common Typing Tips

**Tip 1: Type foreign words in romaji and let the IME decide.** When you type `koohii`, modern IMEs are smart enough to suggest コーヒー in the candidates list because they know it is a common loanword. You do not always need to reach for F7.

**Tip 2: Use F7 for words the IME does not recognize.** If you type a foreign name or technical term and the IME offers hiragana or strange kanji, press F7 to force katakana output.

**Tip 3: Hyphen for ー, not doubled vowels.** In katakana, always use the hyphen key for long vowels. Typing `koohii` will let the IME convert correctly, but when the text appears as katakana, the long vowel mark ー is inserted, not repeated vowels.

**Tip 4: Check your function key behavior on Mac.** If Control+K is not working in a particular app (some apps intercept Control+K for other functions), try Fn+F7 instead, or switch to katakana mode from the menu bar.

**Tip 5: Half-width katakana (F8) is almost never what you want.** Half-width katakana (ｺｰﾋｰ) is a legacy character set used in old systems. For normal use, always use full-width katakana (コーヒー) from F7 or Control+K.

**Tip 6: Use the candidate window for ambiguous sounds.** For words like ジュース (juusu, juice), type `juusu` and press Space to see all candidates. The katakana version will usually appear alongside kanji options.

---

Katakana input comes down to a handful of reliable techniques: the F7 shortcut on Windows, Control+K on Mac, the hyphen for ー, double consonants for ッ, and specific romaji combinations for foreign sounds. Master these and you will handle any katakana word — from everyday loanwords to obscure technical terms — without breaking your typing flow.
